Challenge Javascript 12

Visible dans la console

Un tableau contient les températures maximum des prochains jours en entrée du programme.

Pour chaque température ex [17, 21], afficher dans la console “17 degrés dans 1 jour”, “21 degrés dans 2 jours”.

Données de test : [17, 21, 23] et [12, 5, -5, 0, 4]
