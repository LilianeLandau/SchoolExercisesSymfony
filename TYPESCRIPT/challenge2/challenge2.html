Challenge 2 - TypeScript

Enoncé

Résultat visible dans la console

Voici une liste d'animaux et d'actions possibles sur ceux-ci :

  1. Chat européen (noir)
  2. Chat chartreux
  3. Chien Terre-Neuve (noir)
  4. Chien Moon Moon
  5. Mésange
  6. Merle (noir)
  7. Thon
  8. Requin
  9. Asticot

Et les actions possibles :

  1. On peut photographier tous les animaux
  2. Tous les chats peuvent miauler
  3. Tous les chiens peuvent aboyer
  4. Tous les oiseaux peuvent voler
  5. Tous les poissons peuvent nager
  6. On peut caresser tous les animaux à 4 pattes
  7. On peut nourrir tous les animaux de couleur noire

Il faut :

  1. créer 7 fonctions (correspondants aux 7 actions), prenant chacune le bon type de paramètre
  2. Créer au-dessus de ces 7 fonctions les types correspondants
  3. Mettre le code dans un fichier noe.ts

Les critères de validation sont : les interfaces sont utilisées, les classes sont utilisées, l'héritage est utilisé, il y a 7 fonctions, les fonctions ne doivent avoir qu'un seul paramètre, les fonctions prennent en paramètre les bons animaux
